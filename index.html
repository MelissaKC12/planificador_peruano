ℹ️ Guía rápida
1. Pega tu API Key de Gemini y la URL de tu Apps Script arriba
2. Toca "Guardar configuración" → "Probar conexión"
3. Si la conexión es exitosa, ve a Perfil y llena tus datos
4. Toca "Generar plan" — Gemini usará tus recetas reales de Sheets
5. Registra tus comidas diarias en Adherencia
6. Pesa tus medidas cada lunes en Medidas
⚠️ Configura tu API Key y URL de Sheets en la pestaña Config antes de continuar.
